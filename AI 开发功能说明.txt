# ESP32-S3 打印服务器 - AI 开发功能说明

## 项目概述
基于 ESP-IDF v5.5.3 的 USB 打印服务器项目，支持 WiFi 连接、Web 管理、NTP 时间同步、网络打印等功能。

## 开发环境
- ESP-IDF: v5.5.3
- Python: 3.13
- CMake: 3.30.2
- 工具链：xtensa-esp-elf 14.2.0
- Flash: 16MB
- PSRAM: 8MB

   - 移除重复的空配置检查逻辑
   - 简化 URL 构建流程

2. **安全原则**:
   - 对于可能未初始化的字符串，永远只检查首字符
   - 避免使用任何需要遍历整个字符串的函数（strlen, strnlen, strcmp 等）
   - 优先使用数组索引检查

**修改文件**:
- main/web_hook.c - 移除 strnlen 调用，简化安全检查

## 技术要点

### Web 资源嵌入流程
1. CMake 使用 EMBED_TXTFILES 嵌入 Web 文件
2. 编译器自动生成 _binary_xxx_start/_end 符号
3. web_resources.c 使用这些符号访问嵌入数据
4. 启动时复制到 PSRAM（使用 heap_caps_malloc）
5. Web 服务器从 PSRAM 直接读取

### 分区表配置
```
NVS:      64KB   (0x9000)
otadata:  8KB    (0x19000)
phy_init: 4KB    (0x1B000)
app0:     8056KB (0x20000)
app1:     8056KB (0x800000)
```

### PSRAM 使用
- 使用 MALLOC_CAP_SPIRAM 分配 PSRAM 内存
- Web 资源全部存储在 PSRAM
- 提高 Web 服务器访问速度
- 减少 Flash 读写次数

### USB Host 数据传输
- 使用 usb_host_transfer_alloc 分配传输
- 使用 usb_host_transfer_submit 提交异步传输
- 使用 ulTaskNotifyTake 等待完成
- 支持大数据量连续传输
